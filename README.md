# Vue Data Filter Table

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Go into backend folder
```
cd backend/
```

### Initialize new node project
```
npm init -y
```

### Install Express
```
npm install express
```

### Install CORS
```
npm install cors
```

### Run backend express api (http://localhost:8000/api/tasks)
```
node server.js
```

### Compiles and minifies for production
```
npm run build
```
